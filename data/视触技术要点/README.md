# 视触技术要点学习框架

## 概述
本目录用于系统学习和研究视触传感器（Vision-Based Tactile Sensor, VBTS）技术，包括学术研究、产业发展、技术要点和应用场景。

## 目录结构
```
视触技术要点/
├── papers/              # 学术论文和研究资料
├── concepts/            # 核心概念和技术原理
├── companies/           # 相关公司和企业分析
├── universities/        # 研究机构和高校
├── references/          # 参考资料和文档
├── applications/        # 应用场景和案例
└── market_analysis/     # 市场分析和产业链
```

## 学习目标
1. **学术研究**：收集和整理视触传感器相关学术论文
2. **技术原理**：理解视触传感器的工作原理和核心技术
3. **产业分析**：了解产业链上下游、主要企业和市场趋势
4. **应用场景**：探索视触传感器在机器人、医疗、工业等领域的应用
5. **技术储备**：形成系统的技术知识库和学习材料

## 任务分解
根据GitHub Issue #2要求，学习方向包括：
1. 网上尽可能找到相关的论文
2. 公开的资料
3. 相关的技术产业的上下游
4. 目前行业能做这个方向的学校、公司等并进行罗列
5. 拆成足够细的可以执行的任务
6. 然后形成技术储备
7. 输出一份学习的报告

## 研究方法
- **学术搜索**：使用arXiv、Google Scholar、知网等平台
- **产业调研**：分析行业报告、企业官网、专利信息
- **技术分析**：研究核心专利、技术路线、创新点
- **趋势预测**：基于现有资料预测技术发展方向
