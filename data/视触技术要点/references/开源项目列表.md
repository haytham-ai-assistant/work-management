# 视触传感器开源项目列表

## 概述
本文档整理视触传感器(Vision-Based Tactile Sensor, VBTS)相关的开源项目，包括硬件设计、软件算法、数据集和仿真工具等。

## 1. 开源传感器硬件

### 1.1 DIGIT触觉传感器
- **项目名称**: DIGIT: A Low-Cost Compact High-Resolution Tactile Sensor
- **开发者**: Facebook AI Research (现Meta AI)
- **开源时间**: 2020年
- **技术特点**:
  - 低成本紧凑设计
  - 高分辨率触觉成像
  - 易于制造和组装
  - 标准化接口
- **硬件规格**:
  - 尺寸: 紧凑型设计
  - 分辨率: 高空间分辨率
  - 接口: USB连接
  - 成本: 约$15（不含相机）
- **开源内容**:
  - 3D打印文件
  - 电路设计
  - 组装说明
  - 校准软件
- **GitHub仓库**: https://github.com/facebookresearch/digit
- **论文链接**: https://arxiv.org/abs/2004.00096
- **应用领域**: 
  - 机器人操作研究
  - 触觉感知算法开发
  - 教育研究平台

### 1.2 TacTip系列
- **项目名称**: TacTip: Soft Optical Tactile Sensors
- **开发者**: Bristol Robotics Laboratory
- **技术特点**:
  - 仿生设计，模仿人类指尖
  - 3D打印生物形态结构
  - 多种尺寸和形状变体
- **硬件规格**:
  - 材料: 软性材料
  - 传感原理: 内部标记点跟踪
  - 输出: 3D接触形状
- **开源内容**:
  - 3D打印模型
  - 制造指南
  - 数据采集软件
- **研究论文**: 多篇相关学术论文
- **应用领域**: 
  - 机器人触觉感知研究
  - 物体识别和操作
  - 触觉传感器设计研究

### 1.3 GelSight开源版本
- **项目名称**: OpenGelSight
- **开发者**: 社区驱动项目
- **技术特点**:

## 3. 开源数据集

### 3.1 DIGIT触觉数据集
- **数据集名称**: DIGIT触觉图像数据集
- **数据内容**:
  - 多种物体的触觉图像
  - 不同接触条件下的数据
  - 标注信息（物体类别，姿态等）
- **数据规模**: 数万张触觉图像
- **数据格式**: 图像文件 + 标注文件
- **获取方式**: 公开下载
- **应用领域**:
  - 物体识别算法训练
  - 触觉感知研究
  - 机器学习模型验证

### 3.2 GelSight数据集
- **数据集名称**: GelSight触觉数据集
- **数据内容**:
  - 高分辨率触觉图像
  - 3D几何信息
  - 力分布数据
- **数据特点**:
  - 高质量数据
  - 多种实验条件
  - 详细标注
- **应用领域**:
  - 3D重建算法研究
  - 力估计模型训练
  - 传感器性能评估

### 3.3 TacTip数据集
- **数据集名称**: TacTip触觉数据集
- **数据内容**:
  - 仿生触觉传感器数据
  - 不同物体和表面数据
  - 动态接触序列
- **数据特点**:
  - 仿生传感器数据
  - 时间序列信息
  - 多模态数据
- **应用领域**:
  - 动态触觉感知研究
  - 生物启发算法开发
  - 机器人操作学习
